ruby_fizz_buzz
==============

An implementation & beginner rspec test of:

1. `Integer#to_fizzbuzz`

2. a `FizzBuzz` class, used to output 1 - 100 according to FizzBuzz rules

Example usage
==============

`15.to_fizzbuzz  
=> "FizzBuzz"`

`FizzBuzz::to_s` exists, so the following works:

`puts FizzBuzz  
1  
2  
Fizz  
4  
Buzz  
Fizz  
...`

Run the tests
==============

`rspec`

Currently the 18 tests pass.
